# JavaScript Variables — Team Exercise

A beginner-friendly group exercise to practise declaring variables, using template literals, and printing output with `console.log()` in JavaScript.

---

## 👥 Team Roles

This task is split across 4 team members. Each person works on their own exercise independently.

---

## 👤 Person 1 — Quote Printer

**Task:** Store a quote and an author name in two separate variables, then print them formatted.

**Expected Output:**
```
"Your quote here" - Author Name
```

**Concepts covered:** `const`, string variables, `console.log()`

---

## 👤 Person 2 — Biography Builder

**Task:** Create four variables — `firstName`, `lastName`, `country`, and `job` — and use them to print a biography sentence.

**Expected Output:**
```
Andrew is a developer from India.
```

**Concepts covered:** `const`, multiple variables, template literals

---

## 👤 Person 3 — Profile Card Generator

**Task:** Create variables for `name`, `age`, `country`, and `hobby`, then display a neatly formatted profile card.

**Expected Output:**
```
====================
Name : Solex
Age : 34
Country : Nigeria
Hobby : Coding
====================
```

**Concepts covered:** `const`, mixed data types (string & number), formatted output

---

## 💡 Key Concepts Reminder

| Concept | Example |
|---|---|
| Declare a variable | `const name = "Solex";` |
| Template literal | `` `Hello, ${name}!` `` |
| Print to console | `console.log("Hello");` |
| Number variable | `const age = 34;` |

---

## 🚀 How to Run

1. Create a `.js` file (e.g. `exercise.js`)
2. Write your code
3. Run it in the terminal:
```bash
node exercise.js
```
Or paste it directly into your browser's **DevTools Console** (F12).
